# Project Instructions

Project: ERT - Shared Apartment Management System

## Source of truth
Use the docs folder and current codebase as the source of truth.

## Current stage
Frontend only.
No database yet.
Use mock data only.

## Design rules
- Hebrew RTL
- mobile-first
- clean modern UI
- primary color: #2EC4B6
- background: #FFFDF7
- white cards

## Development rules
- Do not invent features not in the docs
- Do not rewrite unrelated files
- Keep one folder per page under src/pages

## QA rules
When reviewing code:
- compare implementation against docs
- check mobile-first UX
- check natural Hebrew copy
- check RTL consistency
- report bugs and missing requirements clearly
